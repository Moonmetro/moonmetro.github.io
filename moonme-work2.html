Lovely Fat

2020.3-5 / 10th Founder Type Design Competition

I began to do this project during the quarantine period in 2020, attempting to create a glyph that was lovely, a little bit fat and also elegant. When drawing every glyph, I would imagine it moving freely, like fish in the water. Finally good news: Lovely Fat got Merit Award in 10th Founder type design competition.
The project only contains 32 characters in total, so it is still quite limited and is not able to be used in various occasions.
